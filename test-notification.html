What the notification looks like:

Domo Cookie Clearer

431 error detected on test-customer.domo.com. Click the extension icon to clear cookies.

How to Test the Actual Notification:

  1. Make sure the extension is loaded in Chrome
  2. Click the extension icon in your Chrome toolbar
  3. Click the "Test Notification" button in the popup
  4. A system notification should appear (usually in the top-right corner on Mac, bottom-right on Windows)
Note: System notifications appear outside the browser window. They look like native OS notifications with:
  • The extension icon (48x48 cookie icon)
  • Title: "Domo Cookie Clearer"
  • Message about the 431 error
  • They appear in your system's notification center

If notifications don't appear:

  1. Check Chrome notification settings:
    • Go to chrome://settings/content/notifications
    • Make sure notifications are allowed for Chrome
  2. Check system notification settings (macOS/Windows)
  3. Open Chrome DevTools Console (F12) and check for errors
  4. Check the extension's service worker console:
    • Go to chrome://extensions
    • Find "Domo Cookie Clearer"
    • Click "service worker" link to open the console
    • Look for [431 Detection] log messages
Debugging: Open the extension's service worker console to see detailed logs when a 431 error is detected. The logs will show:
  • When a 431 error is detected
  • Domain validation results
  • Notification creation status
  • Any errors that occur
